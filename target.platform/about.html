This examples demonstrates some features of Spring Integration within an OSGi environments.

  • eclipse.org (Equinox, Eclipse Link)
  • springframework.org (for Spring projects)
  • SpringSource Enterprise Bundle Repository (for most other bundles - great resource btw)
